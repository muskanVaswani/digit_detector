# Digit Detector

A basic real-time digit recognition system built using Python, OpenCV, and TensorFlow/Keras. The model is trained on digit image data and predicts digits captured through a webcam feed. The project includes image preprocessing, model training, and live digit prediction using computer vision techniques.

## Features

* Real-time webcam digit detection
* Image preprocessing using OpenCV
* CNN-based digit classification
* Live prediction with confidence score display
* Basic deep learning project

## Tech Stack

* Python
* OpenCV
* TensorFlow / Keras
* NumPy
* Pickle
